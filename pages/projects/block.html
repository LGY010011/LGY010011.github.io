Block : 대외 활동 관리 서비스

" UMC 6th Master 코스 프로젝트로 대학생의 대외 활동 관리 서비스 "

대외 활동 관리 서비스 : 사용자가 지원 가능한 공모전 및 대외 활동 소식을 전해줘요
팀 매칭 서비스 : 직접 원하는 사람에게 하트를 날려요
후기 서비스 : 활동 참여자들의 후기를 볼 수 있어요

피그마 이미지

사용 기술

🖇Spring
🖇JPA
🖇MySQL
🖇AWS
🖇IntelliJ

담당한 기능

BE : API 구현 및 서버 관리

공모전 후기 구매

  • (상품 금액 - 사용할 포인트)의 차액을 카카오페이로 결제
  • 카카오페이 API 연동
  • 이미 결제했던 리뷰인 경우 예외 처리
  • 결제 시 포인트 차감 및 리뷰에 추가
  • 포인트 사용 내역 업데이트
  • 포인트 부족 시 예외 처리

공모전 상세 후기 출력

  • 결제하지 않은 경우: PAY400
  • 리뷰가 contest에 속하지 않을 경우: REVIEW404
  • 본인이 작성한 후기일 경우 정상 출력

기타 기능

  • 잔여 포인트 출력
  • 최근 5개 포인트 상세 내역 출력
  • 프로필 이미지 등록 및 삭제
    • 파일 선택 안 함 → 기존 프로필 사진 삭제
    • 파일 선택 → 새로운 프로필 등록
