동아리 관리자 계정 인증

  • 각 동아리는 하나의 관리자 이메일
  • 자신의 이메일과 동아리명을 담아 요청을 보내면,
    • 서버의 데이터와 일치할 경우 관리자 계정으로 인증
    • 일치하지 않은 경우 NOT_MANAGER 반환

동아리 인증 요청 및 승인

  • 동아리 이름과 자신의 이메일, 학번을 담아 요청
  • 관리자 계정일 경우, 동아리 승인 요청 리스트를 확인 가능
  • 승인 혹은 거절 처리 가능
  • 거절 당할 경우, db에서 삭제되어 재요청이 가능하도록 처리
