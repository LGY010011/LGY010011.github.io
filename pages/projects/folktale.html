담당 파트

은혜갚은 까치

플레이 규칙

  • 사용자는 까치를 움직여 구렁이와 빨간 종을 피해 파란 종을 5번 울리면 Game Clear.
  • 까치의 하트가 다 소진될 경우 Game Over.
  • 빨간 종은 까치를 쫓아오거나 랜덤한 위치에 스폰되며 일정 시간 후 폭발.
  • 구렁이는 1의 데미지를 주며, 빨간 종은 0.5의 데미지를 줌.

핵심 구현 사항

  1. ESC 일시정지 기능 구현:
    • 플레이어가 게임을 중단하고 다시 재개할 수 있도록 ESC 키로 일시 정지 기능 추가.
  2. 플레이 시간에 따른 화면 색상 변화 구현:
    • 게임 진행 시간에 따라 화면 색상이 점진적으로 변화.
    • 일시 정지 시간이 포함되지 않아 실제 플레이 시간에 맞게 반영.
  3. 까치의 위치에 따른 구렁이 움직임 구현:
    • 구렁이가 까치의 현재 위치를 추적하여 전략적 요소를 추가.
    • 플레이어의 이동에 따라 난이도를 조절.
  4. 2초 전 까치의 위치에서 폭발하는 빨간 종 구현:
    • 빨간 종은 2초 전 까치의 위치에서 폭발하도록 설계.
    • 폭발 전 깜빡임 애니메이션과 폭발 중 데미지 활성화를 구현.
  5. 데미지 이후 3초간 무적 상태 활성화:
    • 플레이어가 추가적인 피해를 즉시 받지 않도록 보호.
