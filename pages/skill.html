Front

2024년 하반기, 웹프로그래밍 강의와 UMC 7th Web 챌린저 활동을 통해 프론트엔드 공부를 본격적으로 시작했다. 기초 HTML, CSS, 그리고 JavaScript를 활용해 웹의 기본 구조와 동작 방식을 이해하며, 사용자 인터페이스(UI) 구현에 대한 감각을 키웠다.

웹프로그래밍 강의에서는 웹 페이지 제작의 기초를 다지고, 자바스크립트를 활용해 동적인 기능을 추가하는 방법을 익혔다. 이를 통해 작은 프로젝트를 통해 기본적인 웹 애플리케이션을 구축할 수 있는 역량을 갖추게 되었다.

UMC 7th Web 챌린저 활동에서는 최신 프론트엔드 프레임워크인 React와 CSS-in-JS 라이브러리인 styled-components를 학습했다. 이를 활용해 상태 관리, 컴포넌트 기반 설계, 그리고 재사용 가능한 UI 요소를 구현할 수 있었다. 워크북 실습을 통해 사용자 경험(UX)을 고려한 기능 구현 및 API와의 효율적인 연동 방식에 대해 깊이 있게 이해할 수 있었다.

Back

2024년도 상반기부터 본격적으로 백엔드 개발을 공부하기 시작했다. 처음으로 Spring Boot 프레임워크를 활용하여 RESTful API를 설계하고, 데이터베이스와의 매핑을 통해 비즈니스 로직을 구현했다. 백엔드 개발을 통해 데이터의 흐름과 서버-클라이언트 간의 소통 방식을 깊이 이해하게 되었다.

UMC 6th Server 챌린저 활동을 통해 JPA를 활용한 ORM 기법과 효율적인 쿼리 작성법을 익혔다. 워크북 미션을 통해 다양한 설계 패턴을 실습하며, 유지보수가 쉬운 코드를 작성하는 능력을 길렀다. 특히, 다대다 관계를 비롯한 복잡한 테이블 간 매핑 및 트랜잭션 관리에 대해 학습한 점이 유익했다.

AWS를 활용한 클라우드 환경 구성에도 익숙해졌다. EC2를 사용해 애플리케이션을 배포했으며, S3와 RDS를 통해 스토리지와 데이터베이스 서비스를 효과적으로 연동했다. 이후 CI/CD 파이프라인을 구축하며 자동화된 배포 프로세스를 경험했다. 이러한 과정에서 리눅스 명령어와 인프라 설정의 중요성을 다시 한번 깨달았다.

Game

2024년도 상반기 게임프로그래밍 강의를 통해 C++과 SDL 라이브러리를 활용하여 게임 개발의 기본 원리를 익혔다. 강의에서는 물리 엔진, 충돌 처리, 렌더링 등의 기초를 다루며, 낮은 레벨에서 게임이 어떻게 작동하는지 배울 수 있었다.

팀 프로젝트로 제작한 [설화:folktale]은 고전 설화를 바탕으로 한 어드벤처 게임으로, 캐릭터 이동, 대화 시스템, 간단한 전투 로직 등을 구현했다. 이 프로젝트를 통해 협업 능력뿐 아니라, 게임 기획에서 개발까지의 전 과정을 경험할 수 있었다. 해당 프로젝트는 디지털 이노베이션 행사에서 2위를 수상하며 기술력과 창의성을 인정받았다.

Tool

다양한 개발 도구를 활용하여 효율적인 개발 환경을 구축하고 있다. 프로젝트 관리와 협업에는 Git과 GitHub를 사용하며, IntelliJ와 VS Code는 주요 개발 환경으로 활용 중이다. 특히, IntelliJ는 Spring Boot 기반 프로젝트에서 생산성을 높이는 데 많은 도움을 주었고, VS Code는 프론트엔드 개발 및 경량 작업에서 탁월했다.

데이터베이스 관리에는 JetBrains의 DataGrip을 사용해 SQL 쿼리 작성과 테이블 간 관계를 시각적으로 확인하며, MySQL 데이터베이스를 효율적으로 관리하고 있다. Visual Studio와 Eclipse는 게임 프로그래밍 및 Java 개발에서 각각 사용되며, 프로젝트 요구 사항에 맞게 선택하고 있다.
